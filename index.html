My MERN Project

Mini Social Media Web App

MERN Stack. Click here->

The Mini Social Media web app was created by John Mark S. Ferrer

It is a social media web app that allows users to create profiles, upload images, post content , messages, like, reply and comment on posts, and follow other users.

The app is built using the MERN stack, which includes MongoDB for the database, Express.js for the server, React for the frontend, and Node.js for the backend to handle server-side logic and API requests. User authentication is implemented using JSON Web Tokens (JWT) and web sockets for real-time messaging.

Hosted on Render.com. All free tier.
